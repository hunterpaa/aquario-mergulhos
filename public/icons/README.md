# Ícones do PWA

Você precisa gerar os ícones PNG reais e colocá-los nesta pasta.

## Como gerar

1. Acesse https://realfavicongenerator.net
2. Faça upload de uma imagem quadrada (mínimo 512×512 px) com o logo do aquário
3. Na seção **"Favicon Generator Options"**, ajuste:
   - Background: `#07111f` (dark navy, igual ao tema do app)
4. Clique em **Generate your Favicons and HTML code**
5. Baixe o pacote gerado

## Arquivos necessários nesta pasta

| Arquivo        | Tamanho  |
|----------------|----------|
| `icon-192.png` | 192×192  |
| `icon-512.png` | 512×512  |

Renomeie os arquivos gerados para os nomes acima e cole aqui.

## Sugestão de ícone

Use uma imagem de máscara de mergulho (🤿) ou peixe num fundo azul-marinho (#07111f).
Sites de ícones gratuitos: https://icons8.com · https://flaticon.com
